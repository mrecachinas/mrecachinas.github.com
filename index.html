BeerToCats.com

This started with a desire to learn computer vision techniques and do something cool. I used histogram of oriented gradient (HOG) feature detection and a support vector machine (SVM) classifier to detect solo cups, beer cans, and beer bottles, ultimately covering them up with a random image (like a cat).

While this worked fairly well, it could have been better.

URL Shortener

This was made for fun during my lunchtime one day at work. It uses Node.js and Express.js for the server, Jade for the templating engine, and MongoDB for the database. It's pretty straight forward how it works: you enter a url, it checks the database to see if the url is already in there, otherwise it generates a random alphanumeric string of length 3, checks the database to see if that random string exists, and if not, it inserts the new string and url into the database.

The more interesting part is that I started by generating a string of length 3 as opposed to a larger string. Having a string of length 3 yields (10 + 26 + 26)^3 = 238,328 permutations, which is good enough for my purposes. If that were to be used in practice, to scale, it would start at length 3, then keep track of the number of permutations for length 3, and gradually step up to 4, and so on...
