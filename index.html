BeerToCats.com

This started with a desire to learn computer vision techniques and do something cool. Working with gottagitgit and jb6zb, we used histogram of oriented gradient (HOG) feature detection and a support vector machine (SVM) classifier to detect solo cups, beer cans, and beer bottles, ultimately covering them up with a random image (like a cat).

While this worked fairly well, it could have been better. Some ideas to improve it include training with a ton more samples and using neural nets to achieve a deeper learning.
